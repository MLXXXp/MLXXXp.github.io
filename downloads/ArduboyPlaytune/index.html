Installation Instructions

  1. Download ArduboyPlaytune.zip to a folder on your computer, such as Downloads. The folder you put it in doesn't matter as long as you know how to find the file later. Don't unzip the file.
  2. In the Arduino IDE select from the menus: 
    Sketch > Include Library > Add .ZIP Library...
  3. Locate and select the ArduboyPlaytune.zip file then click on the OK button. The library should now be installed and ready to use.
  4. Once the library is installed, you can delete the ArduboyPlaytune.zip file, if desired.

For more library installation information see
